Gradient Descent

Gradient Descent is a well-known iterative optimization algorithm, widely used in Machine Learning and Deep Learning. It is a way to minimize an objective function f(x) parameterized by a model's parameters x by updating the parameters in the opposite direction of the gradient of the objective function ∇xf(x) w.r.t the parameters.
